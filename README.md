# Level Info
A <cj>simple</c> yet <cj>fully-customizable</c> mod showing <cj>additional information</c> on a <cj>level's page</c>!

## Toggleable features
- <cl>**Requested stars**</c>: display the amount of stars/moons requested by the creator of a level.
- <cl>**Featured Rank**</c>: display a level's featured rank.
- <cl>**Objects**</c>: display the amount of objects in a level.
- <cl>**Objects (with LDM)**</c>: display the amount of objects in a level with Low Detail Mode toggled on.
- <cl>**2-Player Mode**</c>: indicate if a level is intended to be played by 2 players simultaneously.
- <cl>**Game Version**</c>: indicate the game version in which a level was created/updated.
- <cl>**Level Version**</c>: indicate the current version of a level.
- <cl>**Has LDM**</c>: indicate whether a Low Detail Mode exists or not for a level.
- <cl>**Sent**</c>: indicate whether a level has been sent by Moderators or not (uses [SendDB](https://senddb.dev/)).
- <cl>**Level ID**</c>: display the ID of a level.
- <cl>**Original ID**</c>: display the ID of the original level (if it's been copied).
- <cl>**Editor Time**</c>: show how much time the creator(s) of a level spent in the editor.
- <cl>**Editor Time (with copies)**</c>: show how much time the creator(s) of a level spent in the editor (includes previous copies).
- <cl>**Total Attempts**</c>: show how many attempts you spent on a level.
- <cl>**Total Jumps**</c>: show how many times you jumped in a level.
- <cl>**Total Clicks**</c>: show how many times you clicked on your best attempt on a level.
- <cl>**Best Time**</c>: show the best time you did on a level.

## More customizable features
- <cb>**Position**</c>: modify the position of the informational text.
- <cb>**Size**</c>: scale the informational text's size up or down.
- <cb>**Opacity**</c>: make the informational text more or less transparent.
- <cb>**Color**</c>: change the color of the informational text.
- <cb>**Number separator**</c>: choose a separator for big numbers to be more readable.
